## Код в одну строку 

Кусок кода `var temp = 1 < 2 && 5 > 4;` будет встроен в текст

## Много строк кода 

Такой кусок
```

def hello_world():
    print("Hello, World!")
```

Не будет встроен в текст

```python
def hello_world():
    print("Hello, World!")
```
